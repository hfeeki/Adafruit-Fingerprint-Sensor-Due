This is a library for our optical Fingerprint sensor

  Pick one up today in the adafruit shop!
  ------> http://www.adafruit.com/products/751

These displays use TTL Serial to communicate, 2 pins are required to  
interface

IMPORTANT!  Power the Fingerprint Sensor from 3.3V.
            DO NOT use 5V!

This library is for use with the Arduino Due ONLY.

Original library modified for Due by Rick Lesniak.

To download. click the DOWNLOADS button in the top right corner, rename the uncompressed folder Adafruit_Fingerprint_Due. Check that the Adafruit_Fingerprint folder contains Adafruit_Fingerprint_Due.cpp and Adafruit_Fingerprint_Due.h

Place the Adafruit_Fingerprint library folder your <arduinosketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.
